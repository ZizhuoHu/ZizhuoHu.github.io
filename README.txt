Isaac Portfolio Site

How to preview:
1. Open index.html in any browser.

How to edit:
1. Replace the placeholder SVG images in /assets with your own screenshots.
2. Edit project text in /projects/*.html.
3. Edit name, email, and links in index.html, resume.html, and about.html.

How to publish:
- GitHub Pages: upload the folder to a GitHub repository and enable Pages.
- Netlify: drag this folder into Netlify's deploy page.
- Vercel: import the folder/repository and deploy as a static site.
